! Critical blockers — all resolved

Mobile navigation works Hamburger button in the top bar opens a slide-out drawer (academy.js) with full nav + lesson TOC on the lesson page. Backdrop, Escape-to-close, focus-trap-light, and aria-hidden are all wired.
WCAG AA pass :focus-visible ring globally on buttons, links, inputs, video play, password toggle, drawer items. --gray-400 darkened from #A89F96 → #756B62 (5.0:1 on white). Skip-to-main link on every page. Functional emoji replaced with SVG icons + aria-labels.

No invented content Every level name, lesson title, stat number, badge name, and URL is now a {{TEMPLATE_VAR}}. See the variable map below — Antigravity wires these from LearnDash + user meta.

$ Template variable map

Every {{VAR}} in the templates maps to a concrete LearnDash function or WP user-meta key. Pipe defaults ({{VAR|fallback}}) are intentional fallbacks for empty/null.

Template var Source
{{USER_DISPLAY_NAME}}$current_user->display_name
{{USER_INITIAL}}First letter of display name, uppercase

{{XP_NEXT_LEVEL}}computed from your level thresholds table
{{XP_PERCENT}}0–100, (XP_CURRENT − threshold) / (XP_NEXT_LEVEL − threshold) × 100
{{LEVEL_NUM}} / {{LEVEL_NAME}}user meta mw_xp_level + your name lookup. Maharani team owns the level names — none invented here.
